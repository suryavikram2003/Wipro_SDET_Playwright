Playwright Synopsis :
    Building and testing a simple login application using Express and Playwright.
    The webapp serves a login page from the public folder and accepts username and password through a form.
    The backend validates the credentials and returns messages for valid login, invalid login, and input length rules>3.

    The test file uses locators such as #username, #password, #loginBtn, and #msg to interact with the login page.
Used:
    express,dotenv is used to read the PORT value from .env,to avoid exposing in the code.
    The login form uses fetch to send data to /loginwithcreds.
