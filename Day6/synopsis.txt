Flight Booking Data Cleaner:
Problem Statement
Develop a flight data cleaning system that converts unformatted flight booking strings into structured objects for frontend display and processing.

Requirements:
1.Parse flight strings formatted as "CITY_A-CITY_B:PRICE".
2.Extract source city, destination city, and ticket price.
3.Convert price values into numbers.
4.Handle invalid or missing prices by assigning 0.
5.Filter flights with prices between $100 and $500.
6.Sort flights in ascending order based on price.
7.Return the final processed data as a JSON string.
Input Data:
const rawFlights = [
  "London-Paris:150",
  "New York-Tokyo:invalid",
  "Dubai-Mumbai:450",
  "Berlin-Rome:95"
];

Steps:
1.Parsed route and price details using split().
2.Extracted departure and destination cities.
3.Converted price strings into numeric values.
4.Validated prices using isNaN().
5.Created structured flight objects.
6.Filtered flights within the required price range.
7.Sorted flights by lowest price first.
8.Returned formatted JSON output using JSON.stringify().
Output:
[
  {
    "from": "London",
    "to": "Paris",
    "price": 150
  },
  {
    "from": "Dubai",
    "to": "Mumbai",
    "price": 450
  }
]
